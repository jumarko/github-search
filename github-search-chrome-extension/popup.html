Extends GitHub searching capabilities to all your (private) repositories.

To use full text search across all your repositories:

  1. Check options (right click on icon -> Options) and save correct organization name.
  2. Sign in github.com under proper account with access to the configured organization.
  3. Perform full text search by typing "all:some query" into the GitHub search box and pressing enter.
  4. See results!

More info at https://github.com/jumarko/github-search.
